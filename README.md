* Node.js (e.g., v18.x or later)

### Installation & Setup

1. Clone the repository:

   ```bash
   git clone <your-repository-url> # Replace with your actual repository URL
   cd parse-my-bill
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   # yarn install
   ```

3. Set up environment variables:
